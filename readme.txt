==================================================
■登場人物

\0アリス（通称：白アリス）

 フルネームはアリス・ヴァロ・高千穂
 街中で出会ったハーフの少女。 母親がフィンランド人

\1ありす（通称：黒ありす）

 貴方の義妹である少女、以前の経歴は不明
 一緒に暮らす対価として自分の体を差し出した

==================================================
■内容
 このゴーストは「僕のアリス」と「僕のアリス2nd」
 という二部構成になっています。
 
■僕のアリス
 白アリスとの出会い、 黒ありすの登場
 二人との関係を深めていく

 起動数回で「僕のアリス2nd」へ移行可能
 
■僕のアリス2nd
 白アリスと黒ありすの出会い、二人の他愛もない会話
 白アリスと更に関係を深めていく

■操作
cキーでコミュニケート（2nd）
sキーでスキンシップ（2nd）
