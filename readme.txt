==================================================

	ゴースト「僕のアリス2nd」

==================================================
■注意
 このゴーストには18歳未満は使用禁止です。
 このゴーストを利用して如何なる不利益があっても当方は責任を負いません。

==================================================
■登場人物
 アリス（通称：白アリス）
 街中で出会った少女。
 母親がフィンランド人のハーフ。
 
 ありす（通称：黒ありす）
 貴方の義妹である少女。
 貴方と肉体関係を持っている。

==================================================
■内容
 このゴーストは「僕のアリス」と「僕のアリス2nd」
 という二部構成になっています。
 
■僕のアリス
 白アリスとの出会い
 黒ありすの登場
 二人との関係を深めていく
 起動数回で「僕のアリス2nd」へ移行可能
 
■僕のアリス2nd
 白アリスと黒ありすの出会い
 二人の他愛もない会話
 白アリスと更に関係を深めていく

■操作
cキーでコミュニケート（2nd）
sキーでスキンシップ（2nd）

==================================================

■謝辞
 蟻の住む木様作フリーシェル『アリスA』『アリスB』を使用させていただいてます。
 一部にうみほたる様作フォント『にしき的』を使用させていただいでます。
 この場を借りて感謝を申し上げます。
